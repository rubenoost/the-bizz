## Key Decisions

### 2024-12-31 - Product Definition Session
- **Product:** CarShare Tracker - free web app for fair car sharing coordination
- **Core feature:** Fairness algorithm based on `times_driven / (times_driven + times_passenger)` ratio
- **Free tier limits:** 4 members, 2 routes per group (upgrade paths for paid tier)
- **No notifications** in MVP - keep it simple
- **Honour system** for driver selection - users can override suggestions
- **Tech stack:** Next.js + Cloudflare Pages + D1 + Clerk
- **Budget:** $20/month max (should run free on free tiers)
- **Launch:** English only, word-of-mouth
